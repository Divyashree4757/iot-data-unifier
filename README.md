# IoT Data Unifier (Deloitte Simulation)

## 📌 Overview
This project converts two different IoT telemetry data formats into a unified structure.

## 🚀 Features
- Handles multiple JSON formats
- Converts ISO timestamps to epoch milliseconds
- Standardizes nested data structure
- Includes unit tests

## 🛠 Tech Stack
- Python
- JSON
